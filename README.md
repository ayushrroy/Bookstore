# Bookstore
Configure the REST API endpoint and database connection in the appropriate configuration files.
Run the application:
npm run dev
Open your browser and visit http://localhost:8080 to access the Bookstore website.

Usage
Browse the collection of books on the categories page.
Add books to your shopping cart.
Proceed to the checkout to complete your purchase.

Features
Browse Books: Explore a diverse collection of books.
Shopping Cart: Add books to your cart for easy checkout.
Checkout: Complete the purchase of selected books.

Technologies Used
Vue.js: Frontend framework for building user interfaces.
HTML: Markup language for structuring the website.
CSS: Styling language to enhance the visual presentation.
REST API: Communication with the server to fetch and send data.
Pinia Store: State management for Vue.js applications.
SQL: Database for storing and retrieving book information.
